# Installation

## From PyPI

```bash
pip install access_py_telemetry
```

## From conda

```bash
conda install -c conda-forge access_py_telemetry
```

## From source

```bash
git clone https://github.com/ACCESS-NRI/access-py-telemetry.git
cd access-py-telemetry
pip install -e .
```

## Requirements

- Python ≥ 3.10
- IPython ≥ 7.0
